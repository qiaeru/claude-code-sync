Claude Code Sync

Scan your projects and global config, then bundle them into a single password-protected (AES-256) ZIP archive.

Each direct sub-folder is treated as a project. This tool excludes itself.
The archive is named claude-code-sync-<host>-<timestamp>.zip.
After exporting, delete older claude-code-sync-*.zip in the output folder beyond this many.
