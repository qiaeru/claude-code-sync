# Bundled fonts

These font files are bundled so the web UI renders consistently **offline**. Both are licensed under the **SIL Open Font License 1.1** (OFL), which permits bundling and redistribution (including in MIT-licensed software).

| File | Family | Source | License |
| --- | --- | --- | --- |
| `inter-latin-wght-normal.woff2` | Inter (variable, latin) | <https://github.com/rsms/inter> | OFL 1.1 |
| `source-serif-4-latin-wght-normal.woff2` | Source Serif 4 (variable, latin) | <https://github.com/adobe-fonts/source-serif> | OFL 1.1 |

The `.woff2` files were obtained from the Fontsource distribution (<https://fontsource.org>).

> Note: These are open-source approximations of the look of claude.ai. The fonts Anthropic actually uses (Styrene, Copernicus/Tiempos) are proprietary and are **not** bundled here.

The full OFL 1.1 text is available at <https://openfontlicense.org> (and in each upstream repository linked above).
